Lesson 1: What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>
