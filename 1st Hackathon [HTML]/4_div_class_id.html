1. HTML Div

The <div> element is used as a container to group content and structure a webpage.

Element Explanation
<div> Generic container element used to group other HTML elements.
Nested div A div inside another div to structure sections or layouts.

2. HTML Class

The class attribute is used to assign a name to an element for styling or scripting purposes.

Attribute Explanation
class Assigns a class name to an element, can be reused for multiple elements.
Multiple classes One element can have multiple classes separated by spaces.


3. HTML ID

The id attribute assigns a unique identifier to an element for targeting via CSS or JavaScript.

Attribute Explanation
id Unique identifier for an element, should only appear once per page.
Using ID IDs are used for navigation, styling, or JavaScript targeting.

4. HTML Boilerplate

The basic structure of an HTML document containing the required elements for a valid page.

Component Explanation
<!DOCTYPE html> Declares the document type and HTML version.
<html>, <head>, <body> Basic structure of an HTML document containing metadata and content.

5. File Path in HTML

File paths are used to link CSS, JS, images, and other resources in HTML.

Type Explanation
Relative Path Path relative to the HTML file location.
Absolute Path Full path starting from the root or domain.
