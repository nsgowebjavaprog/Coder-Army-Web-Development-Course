1. HTML Lists

HTML provides different types of lists to display content in an organized manner. Lists can be ordered or unordered. They are useful for menus, instructions, steps, and any content that requires grouping.

Element Explanation
<ol> Creates an ordered list where each item is automatically numbered. Example: steps in a tutorial.
<ul> Creates an unordered list with bullet points. Example: shopping list or menu items.
<li> Defines a list item. Used inside <ol> or <ul> to represent individual items.
Nested List A list inside another list for sub-items. Helps create hierarchy or sub-categories in content.

2. HTML Tables

Tables are used to organize and display data in rows and columns. They are especially useful for structured data such as schedules, pricing, comparisons, and reports.

Element / Attribute Explanation
<table> Defines a table element that contains rows and columns.
<tr> Defines a table row that holds table cells.
<th> Defines a table header cell. Text is bold and centered by default.
<td> Defines a table data cell. Regular cell used to hold content.
rowspan Merges a cell vertically across multiple rows. Useful for grouping similar rows.
colspan Merges a cell horizontally across multiple columns. Useful for headings spanning multiple columns.
