HOW Does Internet Work

Basic Networking Concepts

Topic Notes
How Does Internet Work The Internet is a global network of computers connected via routers, switches, and standardized protocols. It allows communication, sharing of resources, and access to web services worldwide.
IP Address A unique number assigned to each device on a network. IPv4 uses 32-bit addresses, while IPv6 uses 128-bit addresses for a larger address space.
IPv6 The latest version of the IP protocol. Supports a huge number of addresses and improved routing efficiency, replacing IPv4 gradually.
MAC Address A hardware identifier assigned to network interfaces. Used to uniquely identify devices at the data link layer of the network.

Networking Tools and Applications

Topic Notes
How to Find MAC Address On Windows: ipconfig /all. On Linux/Mac: ifconfig or ip addr. Shows the hardware address of network interfaces.
Port Numbers Numbers used by applications to identify services (e.g., HTTP = 80, HTTPS = 443, FTP = 21). Allows multiple services on the same IP.
Three Practical Categories of Networking 1. LAN (Local Area Network): Small network in a building or home.
2. MAN (Metropolitan Area Network): Covers a city or campus.
3. WAN (Wide Area Network): Covers large geographical areas, including the Internet.
DNS (Domain Name System) Converts domain names (like google.com) into IP addresses. Makes it easier for humans to access websites without remembering numeric addresses.
